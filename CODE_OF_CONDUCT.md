# Code of Conduct

This repository is maintained as a technical, professional project.
All interactions should stay respectful, constructive, and focused on improving the codebase.

## Expectations

When participating in this project, please:
- Keep discussions technical and relevant to the project
- Be respectful and constructive in issues, pull requests, and reviews
- Provide clear context when reporting bugs or proposing changes
- Assume good intent in technical disagreements

## Unacceptable Behavior

The following behavior is not acceptable:
- Harassment, hate speech, or discriminatory language
- Personal attacks, insults, or hostile comments
- Repeated off-topic discussions or low-effort spam
- Arguing in bad faith or derailing technical conversations

## Maintainer Discretion

This is a personally maintained project.
The maintainer reserves the right to:
- Remove comments, issues, or pull requests that do not align with these guidelines
- Close discussions that are unproductive or disruptive
- Decline contributions that do not fit the projectâ€™s direction

## Scope

This Code of Conduct applies to all project spaces, including:
- GitHub issues
- Pull requests
- Code reviews
- Repository discussions

By contributing to this repository, you agree to follow these guidelines.
